# Deployment

## Vercel

1. Push to GitHub:
   ```bash
   git add .
   git commit -m "feat: complete Astro 8-bit website"
   git push origin feature/astro-8bit-website
   ```

2. Create PR and merge to main branch

3. Vercel will automatically deploy on push to main

## Preview

Run locally:
```bash
npm run preview
```

Open http://localhost:4321
